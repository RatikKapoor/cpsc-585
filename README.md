## Driving Controls

| Action                | Keyboard Control    | Xbox Control     |
|-----------------------|---------------------|------------------|
| Accelerate            | W                   | RT               |
| Brake/Reverse         | S                   | LT               |
| Steer                 | A/D                 | LJS              |
| Fire weapon           | Spacebar            | A                |
| Toggle pause/menu     | ESC                 | START            |


**Debug Only Controls**
| Action                      | Keyboard Control    | Xbox Control     |
|-----------------------------|---------------------|------------------|
| Hot reload vehicle tuning   | R                   | UP               |
| Reset game                  | Backspace            | Y                |
