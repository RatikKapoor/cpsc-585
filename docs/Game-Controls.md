# Game Controls

*This document provides a concrete overview of different controls used by the game.*



| Action                | Keyboard Control    | Xbox Control     |
|-----------------------|---------------------|------------------|
| Accelerate            | W                   | RT               |
| Brake                 | S                   | LT               |
| Reverse               | Left shift          | LB               |
| Steer                 | A/D                 | LJS              |
| Toggle pause/menu     | ESC                 | START            |
